\doxysubsection{Detailed Description}
Main File for extracting \& sorting Usine Nouvelle\textquotesingle{}s ranking. 

\begin{DoxyAuthor}{Author}
Miguel Guzman (\href{mailto:miguel.guzman@imt-atlantique.net}{\texttt{ miguel.\+guzman@imt-\/atlantique.\+net}})
\end{DoxyAuthor}
This file contains the main function that extracts HTML code from Usine Nouvelle\textquotesingle{}s schools of engineering ranking website into a text file. The text file is read to retrieve the information regarding the schools ranking and this information is deposited inside a data structure that can be sorted given user-\/defined sorting parameters. The sorted list of schools is printed into a text file for the user. \begin{DoxySeeAlso}{See also}
